උසස් පෙළ ICT MCQ ප්‍රශ්න 40

විවරණ සහිත බහුවරණ තේරීම් ප්‍රශ්න

කොටස 1: පරිගණක මූලික කරුණු
1 පහත සඳහන් වලින් තොරතුරු පද්ධතියක මූලික කාර්යයන් වන්නේ කුමක්ද?
A. ආදානය, සැකසීම, ප්‍රතිදානය, ගබඩා කිරීම
B. හාඩ්වෙයාර්, මෘදුකාංග, දත්ත, පරිශීලකයන්
C. විද්‍යුත්, ජාලකරණය, ආරක්ෂාව, උපස්ථර
D. සැළසුම්කරණය, ක්‍රියාත්මක කිරීම, පාලනය, මුදල්
පිළිතුර: A. ආදානය, සැකසීම, ප්‍රතිදානය, ගබඩා කිරීම
පැහැදිලි කිරීම: තොරතුරු පද්ධතියක මූලික කාර්යයන් වන්නේ ආදානය (දත්ත ඇතුළත් කිරීම), සැකසීම (දත්ත පරිවර්තනය කිරීම), ප්‍රතිදානය (තොරතුරු ලබා දීම) සහ ගබඩා කිරීම (දත්ත සුරැකීම) යන කාර්යයන් ය.
2 බිටයක් (Bit) හඳුන්වන්නේ:
A. 8 bytes එකකි
B. Binary digit එකකි (0 හෝ 1)
C. 1024 kilobytes එකකි
D. තොරතුරු ඒකකයකි
පිළිතුර: B. Binary digit එකකි (0 හෝ 1)
පැහැදිලි කිරීම: බිටය (Bit) යනු Binary Digit යන්නෙන් ආ පදයකි. එය පරිගණකයේ කුඩාම දත්ත ඒකකය වන අතර 0 හෝ 1 ලෙස අගයන් ගනී. 8 bits = 1 byte, 1024 bytes = 1 kilobyte වේ.
3 1 Gigabyte (GB) සමාන වන්නේ:
A. 1000 MB
B. 1024 MB
C. 1000 KB
D. 1024 KB
පිළිතුර: B. 1024 MB
පැහැදිලි කිරීම: පරිගණක මතක මිනුම් ඒකක ද්විමය පද්ධතිය මත පදනම් වේ. එබැවින් 1 GB = 1024 MB, 1 MB = 1024 KB, 1 KB = 1024 bytes වේ. දශම පද්ධතියේ 1000 වෙනුවට ද්විමය පද්ධතියේ 1024 (2¹⁰) භාවිතා වේ.
4 පහත සඳහන් කුමන උපාංගය ආදාන උපාංගයක් නොවේද?
A. Scanner
B. Keyboard
C. Printer
D. Mouse
පිළිතුර: C. Printer
පැහැදිලි කිරීම: ප්‍රින්ටරය යනු ප්‍රතිදාන උපකරණයකි. එය පරිගණකයෙන් තොරතුරු ලබාගෙන එය කඩදාසියක මුද්‍රණය කරයි. Scanner, Keyboard, Mouse යනු ආදාන උපකරණ වේ.
5 CPU හි ALU කොටසෙන් සිදු කරන්නේ:
A. දත්ත ගබඩා කිරීම
B. ගණිතමය හා තාර්කික ක්‍රියා
C. ආදාන පාලනය
D. ප්‍රතිදාන පාලනය
පිළිතුර: B. ගණිතමය හා තාර්කික ක්‍රියා
පැහැදිලි කිරීම: ALU (Arithmetic Logic Unit) යනු CPU හි ගණිතමය (එකතු කිරීම, අඩු කිරීම, ගුණ කිරීම, බෙදීම) සහ තාර්කික (AND, OR, NOT) ක්‍රියා සිදු කරන කොටසයි.
කොටස 2: මෘදුකාංග හා යෙදුම්
6 පහත සඳහන් කුමන එක පද්ධති මෘදුකාංගයකි?
A. MS Word
B. Windows 10
C. Adobe Photoshop
D. Google Chrome
පිළිතුර: B. Windows 10
පැහැදිලි කිරීම: පද්ධති මෘදුකාංග යනු පරිගණක දෘඩාංග සහ යෙදුම් මෘදුකාංග අතර මඟ පෙන්වන මෘදුකාංග වේ. Windows 10 යනු මෙහෙයුම් පද්ධතියක් වන අතර එය පද්ධති මෘදුකාංගයකි. අනෙක් ඒවා යෙදුම් මෘදුකාංග වේ.
7 Open Source මෘදුකාංගයේ ප්‍රධාන ලක්ෂණය වන්නේ:
A. මිල අධිකයි
B. මූල කේතය ලබා ගත හැකියි
C. ලිපි ලේඛන සඳහා පමණි
D. අන්තර්ජාලය අවශ්‍ය නැත
පිළිතුර: B. මූල කේතය ලබා ගත හැකියි
පැහැදිලි කිරීම: විවෘත මූලාශ්‍ර (Open Source) මෘදුකාංගවල ප්‍රධාන ලක්ෂණය වන්නේ ඒවායේ මූල කේතය පරිශීලකයින්ට ලබා ගත හැකි වීම, වෙනස් කිරීමට හැකි වීම සහ බෙදාහැරීමට හැකි වීම ය. Linux, Apache, MySQL යනු විවෘත මූලාශ්‍ර මෘදුකාංග උදාහරණ ය.
8 Database Management System (DBMS) භාවිතා කරන්නේ:
A. වීඩියෝ සංස්කරණය සඳහා
B. දත්ත සංවිධානය හා කළමනාකරණය සඳහා
C. ගණිතමය ගණනය කිරීම් සඳහා
D. සංගීත නිර්මාණය සඳහා
පිළිතුර: B. දත්ත සංවිධානය හා කළමනාකරණය සඳහා
පැහැදිලි කිරීම: දත්ත සමුදාය කළමනාකරණ පද්ධතිය (DBMS) යනු දත්ත සංවිධානය කිරීම, ගබඩා කිරීම, ලබා ගැනීම, කළමනාකරණය කිරීම සහ ආරක්ෂා කිරීම සඳහා භාවිතා කරන මෘදුකාංග පද්ධතියකි. MySQL, Oracle, SQL Server යනු DBMS උදාහරණ ය.
9 MS Excel හි Formula එකක් ආරම්භ වන්නේ:
A. #
B. =
C. @
D. $
පිළිතුර: B. =
පැහැදිලි කිරීම: Microsoft Excel හි සූත්රයක් (Formula) ආරම්භ කිරීම සඳහා සමාන ලකුණ (=) භාවිතා කරයි. උදාහරණයක් ලෙස =A1+B1, =SUM(A1:A10) වැනි සූත්ර සමාන ලකුණෙන් ආරම්භ වේ.
10 පහත සඳහන් කුමන දත්ත වර්ගය SQL හි භාවිතා වේද?
A. VARCHAR
B. STRING
C. TEXT256
D. CHARACTER_LONG
පිළිතුර: A. VARCHAR
පැහැදිලි කිරීම: SQL හි VARCHAR (Variable Character) යනු විචල්‍ය දිග ඇති අකුරු දත්ත වර්ගයකි. INT (පූර්ණ සංඛ්‍යා), DATE (දිනය), DECIMAL (දශම සංඛ්‍යා) යනු අනෙකුත් SQL දත්ත වර්ග ය.
කොටස 3: ජාලකරණය හා අන්තර්ජාලය
11 LAN යන්නෙන් අදහස් වන්නේ:
A. Large Area Network
B. Local Area Network
C. Linear Access Network
D. Long Active Network
පිළිතුර: B. Local Area Network
පැහැදිලි කිරීම: LAN (Local Area Network) යනු කුඩා භූගෝලීය ප්‍රදේශයක් (ගොඩනැගිල්ලක් හෝ කැම්පසයක්) ආවරණය කරන පරිගණක ජාලයකි. WAN (Wide Area Network) විශාල ප්‍රදේශ ආවරණය කරන ජාලයකි.
12 පහත සඳහන් වලින් වලංගු IP address එකක් වන්නේ:
A. 192.168.1.256
B. 192.168.1.1
C. 192.168.256.1
D. 256.168.1.1
පිළිතුර: B. 192.168.1.1
පැහැදිලි කිරීම: IPv4 ලිපිනයක එක් එක් කොටස 0 සිට 255 දක්වා අගයක් ගත හැකිය. 256 යනු වලංගු අගයක් නොවේ. එබැවින් 192.168.1.1 පමණක් වලංගු IP ලිපිනයකි.
13 HTTP යන්නෙන් අර්ථ දක්වන්නේ:
A. Hyper Text Transfer Protocol
B. High Transfer Text Protocol
C. Home Tool Transfer Protocol
D. Hyperlink Text Transmission Process
පිළිතුර: A. Hyper Text Transfer Protocol
පැහැදිලි කිරීම: HTTP (Hyper Text Transfer Protocol) යනු වෙබ් පිටු ලබා ගැනීම සහ යැවීම සඳහා භාවිතා කරන ප්‍රොටෝකෝලයකි. HTTPS යනු ආරක්ෂිත අනුවාදයයි.
14 WWW හි 'W' යන්නෙන් අදහස් වන්නේ:
A. Wireless
B. World
C. Wide
D. Web (World Wide Web)
පිළිතුර: B. World (WWW - World Wide Web)
පැහැදිලි කිරීම: WWW යන්නෙන් අදහස් වන්නේ World Wide Web ය. එය අන්තර්ජාලයේ තොරතුරු ප්‍රවේශය සඳහා භාවිතා කරන සේවාවකි. ටිම් බර්නර්ස් ලී විසින් 1989 දී නිර්මාණය කරන ලදී.
15 Router එකක කාර්යය වන්නේ:
A. දත්ත ගබඩා කිරීම
B. ජාල අතර දත්ත මාර්ගගත කිරීම
C. වයිරස් හඳුනා ගැනීම
D. මුද්‍රණය කිරීම
පිළිතුර: B. ජාල අතර දත්ත මාර්ගගත කිරීම
පැහැදිලි කිරීම: රවුටරයක් ජාල අතර දත්ත පැකට්ටු මාර්ගගත කිරීම සිදු කරයි. එය ජාලවලට සම්බන්ධ වී ඒවා අතර දත්ත හුවමාරුව පාලනය කරයි.
කොටස 4: වෙබ් තාක්ෂණය
16 HTML යන්නෙන් අර්ථ දක්වන්නේ:
A. High Text Markup Language
B. Hyper Text Markup Language
C. Home Tool Markup Language
D. Hyperlinks Text Making Language
පිළිතුර: B. Hyper Text Markup Language
පැහැදිලි කිරීම: HTML (Hyper Text Markup Language) යනු වෙබ් පිටු සෑදීම සඳහා භාවිතා කරන සම්මත මාර්කප් භාෂාවකි. එය වෙබ් පිටුවක ව්‍යුහය නිර්වචනය කිරීම සඳහා විවිධ ටැග් භාවිතා කරයි.
17 CSS භාවිතා කරන්නේ:
A. වෙබ් පිටු නිර්මාණය සඳහා
B. දත්ත සංචිත සඳහා
C. වෙබ් පිටු මෝස්තර කිරීම සඳහා
D. සේවාදායක පාලනය සඳහා
පිළිතුර: C. වෙබ් පිටු මෝස්තර කිරීම සඳහා
පැහැදිලි කිරීම: CSS (Cascading Style Sheets) යනු HTML ලේඛනවල පෙනුම සහ ආකෘතිය විස්තර කිරීම සඳහා භාවිතා කරන භාෂාවකි. එය වර්ණ, අකුරු, පරතරය, පිහිටුම වැනි දේ නිර්වචනය කිරීමට ඉඩ සලසයි.
18 පහත සඳහන් කුමන HTML tag එක paragraph එකක් නිර්මාණය කරයිද?
A. <p>
B. <para>
C. <paragraph>
D. <pg>
පිළිතුර: A. <p>
පැහැදිලි කිරීම: HTML හි <p> ටැගය ඡේදයක් (paragraph) නිර්මාණය කිරීම සඳහා භාවිතා කරයි. <h1> සිට <h6> දක්වා ශීර්ෂ සඳහා, <div> කොටස් සඳහා භාවිතා වේ.
19 JavaScript භාවිතා කරන්නේ:
A. Server-side පමණක් සඳහා
B. Client-side අන්තර්ක්‍රියා සඳහා
C. Database management සඳහා
D. Network security සඳහා
පිළිතුර: B. Client-side අන්තර්ක්‍රියා සඳහා
පැහැදිලි කිරීම: JavaScript යනු ප්‍රධාන වශයෙන් Client-side (Browser) හි ක්‍රියාත්මක වන අන්තර්ක්‍රියාකාරී වෙබ් පිටු නිර්මාණය කිරීම සඳහා භාවිතා කරන ප්‍රග්‍රහණ භාෂාවකි. Node.js වැනි ප්‍රකාශන සමඟ Server-side හි ද භාවිතා කළ හැකිය.
20 Dynamic වෙබ් පිටුවක් යනු:
A. සෑම විටම එකම අන්තර්ගතය පෙන්වන එකයි
B. පරිශීලක ක්‍රියාවන්ට ප්‍රතිචාර දක්වන එකයි
C. පින්තූර නැති එකයි
D. HTML පමණක් ඇති එකයි
පිළිතුර: B. පරිශීලක ක්‍රියාවන්ට ප්‍රතිචාර දක්වන එකයි
පැහැදිලි කිරීම: Dynamic වෙබ් පිටුවක් යනු පරිශීලක ක්‍රියාවන්, දත්ත සමුදාය වෙනස් කිරීම්, හෝ වෙනත් සංසිද්ධි මත පදනම්ව අන්තර්ගතය වෙනස් කරන පිටුවකි. Static පිටු සෑම විටම එකම අන්තර්ගතය පෙන්වයි.
